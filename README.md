# Custom Skills

Shared custom AgentSkill collection for the Resonance/OpenClaw workspace.

## Included skills

Skills are stored under `skills/` and should be portable across compatible agent runtimes.

## Workflow

- Keep each skill self-contained: `SKILL.md` plus optional `scripts/`, `references/`, `assets/`, or `agents/`.
- Keep `SKILL.md` lean and move long examples or implementation notes into `references/`.
- Validate YAML frontmatter before publishing changes.
- When a workflow improves after real use, update the skill and commit the learning.
